# Grupo 14 - TPO API (Aplicaciones Interactivas)

Este repositorio contiene la API REST desarrollada para el Trabajo Práctico Obligatorio (TPO) de la materia **Aplicaciones Interactivas** de la UADE.

## Contexto del Proyecto

La aplicación expone diversos endpoints mediante una arquitectura basada en **Spring Boot**, orientada a gestionar clientes, préstamos y solicitudes de aumento de límites de créditos, así como todo el flujo de cobranzas y cuotas.
